# Cube Rhythm
在Minecraft中实现音游

本项目完全使用Skript及其附属编写，对应服务端内plugins/skript/scripts目录

## 已知问题
1. 性能问题，由于每个Tick要传送一遍所有的音符导致
   （需要比较强劲的电脑！否则加载时间可能过长，面对大量实体时出现卡顿甚至音画不同步等问题）
2. 在没有与Double音符交互时，对准其左键单击会同时击打两次，目前暂无解决方案
3. 偶见吞音符现象：少量音符隐身，但不影响判定，与客户端和服务器延迟有关
4. 音符排列偶见轻微移位
